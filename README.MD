# 👥 Employee Management System

A full CRUD application for employee management built with ExtJS 6 on the frontend and Node.js + Express + PostgreSQL on the backend.

## Technologies
- **Frontend:** ExtJS 6 (via CDN)
- **Backend:** Node.js + Express
- **Database:** PostgreSQL

## How to run locally

### Prerequisites
- Node.js installed
- PostgreSQL installed

### Backend
```bash
cd backend
npm install
# Set up the .env file with your credentials
node server.js
```

### Frontend
Open the `frontend/index.html` file with Live Server or any HTTP server.

## Features
- List all employees
- Create a new employee
- Edit an existing employee
- Delete an employee
